# AGENTS.md

## Commit Messages

Use imperative mood: "add feature", not "added feature".

The subject line should complete the sentence: "If applied, this commit will…"

Always preview the commit message and author, and wait for explicit approval before running `git commit`.

Never include a `Co-Authored-By: Codex` trailer in commit messages.

## Workflow

Use OpenSpec (`/opsx:propose` → review → `/opsx:apply` → `/opsx:archive`) for all changes instead of writing files directly.
